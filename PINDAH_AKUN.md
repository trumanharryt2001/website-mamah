# Catatan pindah akun GitHub + Supabase

Folder ini sudah dibersihkan dari folder `.git`, jadi tidak lagi terhubung ke repo `tottifawwazr`.

Langkah singkat:
1. Buka folder ini di VS Code.
2. Jalankan `npm install`.
3. Buat file `.env.local` di root project.
4. Isi dengan URL dan publishable key dari project Supabase baru.
5. Push/publish ke repo GitHub baru dari akun `trumanharryt2001`.

Format `.env.local`:

```env
VITE_SUPABASE_URL=https://xxxxx.supabase.co
VITE_SUPABASE_PUBLISHABLE_KEY=xxxxx
```
